# Project Title

Real‑Time Coal Stock Analytics for India’s Thermal Power Fleet

## Description

India’s thermal power stations are heavily dependent on coal as their primary fuel source. Ensuring adequate coal stock levels is critical for uninterrupted electricity generation and national energy security. This project focuses on building an analytical framework to monitor, evaluate, and derive insights from daily coal stock data collected across the country.

## Getting Started

### Dependencies

* Python 3.9+
* Libraries: pandas, numpy, matplotlib, seaborn
* OS: Windows 10 or Linux (tested on Ubuntu 20.04)
* Jupyter Notebook (optional, for interactive analysis)

### Installing
* Clone the repository:
```bash
git clone https://github.com/yourusername/coal-stock-analytics.git
cd coal-stock-analytics

### Executing program

```
python analyze_coal_stock.py
```
(Optional) Launch Jupyter Notebook for interactive exploration
```
jupyter notebook CoalStockAnalysis.ipynb
```
## Help
The below file will set up everything automatically.
```
pip install -r requirements.txt 
```
